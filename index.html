AI-Based Mutual Fund Recommendation System

Smart, Personalized & Inclusive Financial Guidance

Presented by: Deekshitha Bhairav

Problem Statement

  • Investing in mutual funds is confusing for many.
  • No personalized financial advice in traditional systems.
  • Physically or cognitively challenged individuals are often excluded.
  • Manual research is time-consuming & unreliable.

Proposed Solution

An intelligent system that recommends mutual funds using machine learning, tailored to:

  • User goals
  • Risk appetite
  • Investment duration

With a user-friendly, inclusive interface and real-time suggestions.

How It Works

Flow:

User Input → Goals, Risk Level → ML Model → Fund Data Analysis → Smart Recommendations

Real-time data from APIs for performance-based results

Key Features

  • AI-powered decision-making
  • Goal-based suggestions
  • Voice-enabled accessibility
  • Live updates via APIs
  • Inclusive design (mobile/web/SMS)

Impact & Use Cases

Empowering all users for smarter financial decisions:

  • Visually impaired (voice support)
  • Hearing impaired (visual cues)
  • Cognitively challenged (step-by-step help)
  • Used by: Banks, fintech apps, government campaigns

Challenges & Limitations

  • Depends on high-quality fund data
  • Needs regular updates
  • User trust in AI suggestions needs time
  • Advanced customization for high-value users

Conclusion

A practical, creative system for smart investing – accessible, inclusive & impactful.

Ready for real-world deployment and future innovations!
